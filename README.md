# 🟣 KeyOne — مساعد المبيعات الذكي على واتساب

**نظام كامل** يحوّل واتساب بزنس إلى آلة مبيعات تعمل 24/7:
- 🤖 يرد على العملاء فوراً بالعربية والإنجليزية
- 🔥 يكشف العميل الجاهز للشراء ويُنبهك على رقمك
- 🧠 يتعلم من ملفاتك (منتجات، أسعار، عروض، سكريبتات)
- 🎯 لا يكرر، لا يتجاوب مع الرسائل العشوائية، ويحترم أوامر "إيقاف"
- 📊 لوحة تحكم احترافية كاملة بهوية بصرية مميزة

---

## 📦 محتوى الحزمة


## ⚡ التشغيل السريع (الاختبار المحلي)

### المتطلبات
- **Node.js 20+** ([تحميل](https://nodejs.org))
- مفتاح Groq API مجاني من [console.groq.com](https://console.groq.com)
- رقم واتساب بزنس

### الخطوات

#### على ويندوز
```batch
1) شغل install.bat   (تثبيت الحزم - مرة واحدة فقط)
2) شغل start.bat     (تشغيل النظام)
```

#### على Linux/Mac
```bash
chmod +x install.sh start.sh
./install.sh
./start.sh
```

ثم افتح المتصفح على: **http://localhost:3000**

البريد الافتراضي: `admin@keyone.local`
كلمة المرور: `admin1234`
> ⚠️ **غيّرها فوراً من صفحة الإعدادات.**

## ⚙️ الإعداد الأساسي

### 1. مفتاح AI (مهم جداً)
- اذهب لـ `الإعدادات` → الصق مفتاح **Groq API** (مجاني)
- أو استخدم OpenAI إن أردت

### 2. ربط واتساب
- اذهب لـ `ربط واتساب` → امسح QR من تطبيق واتساب بزنس
- (الإعدادات → الأجهزة المرتبطة → ربط جهاز)

### 3. تدريب الذكاء على بياناتك
- صفحة `تدريب الذكاء` → أضف:
  - وصف المنتجات والباقات
  - الأسعار والعروض
  - أسئلة شائعة وإجاباتها
  - سكريبتات الإغلاق
  - سياسات الإرجاع/الدفع

النظام يقرأ كل هذه البيانات ويستخدمها في الردود.

### 4. اضبط رقم المالك
- في `الإعدادات` → أدخل رقمك (بدون +)
- ستصلك إشعارات لحظة جاهزية أي عميل للشراء

## ✨ المميزات الكاملة

### ذكاء بيعي حقيقي
- ✅ يفهم العميل من سياق المحادثة (ليس مجرد كلمات مفتاحية)
- ✅ يكتشف نية الشراء فوراً ("موافق", "ابعتلي الرابط", "i agree" → تنبيه فوري)
- ✅ يميّز "الرقم الغلط" بدقة (لا إنذارات كاذبة)
- ✅ يحترم أوامر "إيقاف" / "stop" تلقائياً

### حماية ذكية ضد الحظر
- ⏱️ تأخير عشوائي في الرد (يحاكي الإنسان)
- ⌨️ مؤشر "يكتب الآن…" واقعي
- 📉 حد أقصى للرسائل في الدقيقة
- 🌙 ساعات عمل قابلة للضبط (يرد خارجها برسالة "مغلق")

### واجهة احترافية
- 🎨 هوية بصرية حصرية K1 (بنفسجي كهربائي + halftone)
- 📱 متجاوبة بالكامل (موبايل + تابلت + ديسكتوب)
- 🌙 dark mode دائم
- 🇸🇦 RTL عربي بشكل صحيح
- ⚡ تحديث لحظي للرسائل (WebSocket)

### إدارة العملاء
- 📋 قائمة كل العملاء مع درجة اهتمامهم
- 🏷️ حالات: بارد · دافئ · ساخن · تم البيع · مفقود
- 💬 محادثة ثنائية الاتجاه (يمكنك الرد يدوياً)
- 🚫 قائمة سوداء + إيقاف تلقائي
- 📊 إحصائيات + تصدير CSV
- 🗑️ حذف محادثات (فردي أو جماعي)

### تجميع الرسائل (Debouncing)
- إذا أرسل العميل عدة رسائل سريعة، يجمعها AI ويرد رداً **واحداً** ذكياً عليها كلها
- لا تكرار، لا تناقض، لا "spam" من البوت

## 🛠️ الدعم الفني

عند أي مشكلة، تأكد من:
1. `Node.js 20+` مُثبت (`node -v`)
2. مفتاح AI صالح في الإعدادات
3. رقم واتساب صحيح بصيغة دولية (بدون +)
4. الباك إند يعمل على المنفذ `4000`
5. الفرونت يعمل على المنفذ `3000`

سجلات الأخطاء في:
- Backend: شاشة الـ terminal
- Frontend: console المتصفح (F12)
